Учебный IaC-проект: автоматическое развёртывание сервиса Speedtest в k3s-кластере на Yandex Cloud.

## 🛠️ Как это работает

Процесс развертывания разделен на два этапа: создание облачной инфраструктуры и конфигурация окружения. Ручная настройка виртуальной машины не требуется 

### Стек технологий
* **IaC (Инфраструктура как код):** Terraform (управление ресурсами YC)
* **Конфигурация:** Ansible (настройка ОС, установка k3s и деплой)
* **Оркестрация:** k3s (легковесный Kubernetes)

### Архитектура и этапы развертывания

1. **Инициализация:** Вы вызываете `terraform apply`.
2. **Инфраструктура:** Terraform создает виртуальную машину, в Yandex Cloud.
3. **Связующее звено:** С помощью провайдера `local_file` динамически создается файл `hosts.ini` с актуальным IP-адресом новой VM.
4. **Конфигурация:** Передача сгенерированного инвентаря hosts.ini в Ansible для настройки узла.
5. **Оркестрация:** Ansible устанавливает k3s, клонирует Kubernetes-манифесты и разворачивает сервис Speedtest.
